# TransPulse — Andhra Pradesh Smart Mobility Platform

**The Pulse of Public Transportation**

A production-grade real-time fleet tracking and transport management platform for Andhra Pradesh, covering Andhra Pradesh.

---

## 🚀 Features

### 🗺️ APSRTC Live Tracking
- **Live Bus Tracking**: Real-time GPS positioning with 5-second refresh
- **Andhra Pradesh Coverage**: 4 states, 50+ cities, 25+ strategic routes
- **Mandatory Palasa Hub**: Key North Andhra route hub
- **Dark-Themed Maps**: Custom Leaflet integration with animations

### 📊 Route Intelligence
- **Advanced ETA Engine**: Distance prediction and stop-by-stop timing
- **25 Strategic Routes**: Including:
  - Palasa ↔ Visakhapatnam
  - Plus coastal, pilgrim, and regional corridors

### 👨‍💼 Operations Dashboard
- **Real-Time KPIs**: Animated counters and live metrics
- **Fleet Monitoring**: Active buses and routes
- **Admin Control**: Bus and route management
- **Analytics**: Performance insights

### 👤 User Roles
- **Admin**: Full operations control
- **Driver**: Trip management and telemetry
- **Passenger**: Real-time tracking and ETA

### 📱 Mobile Experience
- **PWA Support**: Install as native app
- **Offline Mode**: Works without internet
- **Responsive Design**: Mobile-first layout
- **Bottom Navigation**: Touch-friendly controls

### 🎨 Premium Design
- **Glassmorphism**: Modern frosted glass effects
- **Dark Theme**: Navy backgrounds with cyan accents
- **Smooth Animations**: Hero pulse, counter animations
- **Professional Typography**: Poppins & Inter fonts

---

## 📍 Regional Coverage

### Phase 1: Live (AP)
**Andhra Pradesh**: Palasa, Visakhapatnam, Rajahmundry, Vijayawada, Nellore, Tirupati, Kurnool, and all major districts
---

## 🛠️ Tech Stack

- **Backend**: Flask + SQLAlchemy + SQLite
- **Frontend**: HTML5, Bootstrap 5, Leaflet.js
- **PWA**: Service Workers, Web App Manifest
- **Auth**: Flask-Login + Werkzeug

